Fetch

Fetch é uma API JavaScript moderna que permite realizar requisições assíncronas a servidores. Ela utiliza promessas, facilitando o manejo de operações de rede sem bloquear a execução do código. A API Fetch é mais limpa e flexível do que o antigo XMLHttpRequest, permitindo que os desenvolvedores façam solicitações GET, POST, PUT e DELETE de forma mais simples e eficiente.
